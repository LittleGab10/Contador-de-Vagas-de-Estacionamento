# Contador-de-Vagas-de-Estacionamento
Aqui teremos o código para o projeto de um contador de vagas em um estacionamento, além do seu projetos rtl completos e exemplos de formas de onda. Feito no quartus em linguagem vhdl e simulado numa FPGA.
​O projeto consiste em um sistema  que monitora 4 vagas de estacionamento . O objetivo é calcular a quantidade de vagas disponíveis , exibindo o total de vagas livres (de 0 a 4) em um display de 7 segmentos e acionando um LED de alerta quando o estacionamento estiver totalmente lotado
